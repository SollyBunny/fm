# fm
A terminal file manager made in C  
Lots of inspiration took from [nnn](https://github.com/jarun/nnn/)  
This project is a WIP
## features
(More like wishlist)  
* Nerd font icons
* FIFO pipe for plugin integration
* Sane default keybinds (GUI ones)
* Grep / Fuzzy search
* Very moddable
* Other stuff :)
